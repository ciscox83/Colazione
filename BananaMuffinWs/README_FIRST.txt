1. Download and install Google Cloud SDK by running the following command in your shell or Terminal:

           curl https://sdk.cloud.google.com/ | bash

2. Restart your shell or Terminal to pick up environment changes.

3. Sign in to Google Cloud Platform using this command:

           gcloud auth login

4. Install the App Engine package for Python using this command:

           gcloud components update gae-python

5. Use the deploy.sh script to deploy

6. Visit:

           banana-muffin.appspot.com
